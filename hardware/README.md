# Hardware

## Components
Pi Pico (RP2040)
GPS Receiver
Toggle switch with LED
Micro-USB to USB-C adapter
Project box

## Soldered connections
Pico GP0 (UART Tx) <-> GPS Rx
Pico GP1 (UART Rx) <-> GPS Tx
Pico 3v3 <-> GPS 3v3
Pico Gnd <-> GPS Gnd

Pico VSys <-> Switch Vcc
Pico GP?? <-> Switch out
Pico Gnd <-> Switch Gnd
